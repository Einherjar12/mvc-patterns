# mvc-patterns
MVC (Model-View-Controller) — архитектурный паттерн, который разделяет приложение на три части:  Модель (Model) — управляет данными и логикой ; Представление (View) — показывает данные пользователю ; Контроллер (Controller) — обрабатывает действия пользователя и связывает модель с представлением.
